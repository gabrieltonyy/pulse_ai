## How to Export Bob Sessions

### From VSCode Bob Extension

1. Open Bob conversation panel
2. Click "Export" button
3. Select "Export as Markdown"
4. Save to `bob_sessions/` directory
5. Name file descriptively (e.g., `session_01_initial_planning.md`)
